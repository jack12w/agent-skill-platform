## 任务 2：提交 Google Search Console

**预计时间**：15 分钟  
**地址**：https://search.google.com/search-console

### 操作步骤

1. 用 Google 账号登录 Search Console
2. 点击「添加资源」，选择「网域」类型（不是「网址前缀」）
3. 输入：`skills.rehomi.com`
4. 系统会生成一条 TXT 记录，**去你的域名 DNS 管理后台添加这条记录**
   - 记录类型：TXT
   - 主机记录：@
   - 记录值：Google 提供的那一串字符
5. 回到 Search Console 点击「验证」
6. 验证成功后，进入「站点地图」→ 输入 `sitemap.xml` → 点击「提交」
7. 进入「网址检查」→ 输入首页 URL `https://skills.rehomi.com/` → 点击「请求编入索引」

### 可能需要问域名商的操作
- DNS 管理后台在哪？（阿里云域名在「阿里云控制台 → 域名 → 解析设置」）
- 注意：DNS 记录生效需要 1~10 分钟，不要急

### 验证标准
- [ ] Search Console 显示「已验证」
- [ ] sitemap 状态为「成功」
- [ ] 首页已请求编入索引

## 任务 3：提交百度搜索资源平台

**预计时间**：15 分钟  
**地址**：https://ziyuan.baidu.com

### 操作步骤

1. 用百度账号登录
2. 点击「添加站点」→ 输入 `skills.rehomi.com`
3. 验证方式选「DNS 验证」（最稳定），同样去 DNS 后台加 TXT 记录

- 或者：直接在 Maker 身份下发帖（不算 Hunt 形式），流量会少一些但也能做

### 验证标准

- [ ] 找到了 Hunter 或确认 Maker 发帖
- [ ] 发布时间已确定为周二/周三
