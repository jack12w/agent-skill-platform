最近做了个东西，叫 **SkillDepot**（技能广场），网址：https://skills.rehomi.com

**它解决什么问题？**

做跨境的朋友都知道，AI Agent 工具越来越多，但每次都得自己搭配提示词、写工作流、调参数。
SkillDepot 想做的事很简单：**把这些成熟的 AI 技能打包成标准格式，放到一个平台上，大家可以上传、下载、分享。**

**目前重点覆盖的场景：**
- 阿里巴巴国际站：Listing 自动优化、RFQ 回复、关键词分析
- 亚马逊：广告优化、Review 管理、选品分析、A+ 内容生成
- Shopify / 独立站：产品描述生成、客服机器人、SEO 优化
- WordPress 电商：WooCommerce 自动化
- 通用跨境：社媒营销、供应链管理、合规风控

**技术栈：**
Next.js 14 + NestJS + PostgreSQL + Docker，一键部署，代码开源。

**一些功能：**
- 上传：ZIP 包 + SKILL.md 元数据，5 分钟发布一个技能
- 下载：一键下载，可直接接入 WorkBuddy 等 AI 工作流平台
- 排行榜：周榜/总榜，个人/团队榜，防刷分机制
- 评论：打分 + 留言，社区筛选高质量技能

**目前没有流量，所以来这里问一下：**
1. 跨境电商的朋友，你们在用 AI 做什么具体的事？
2. 有没有人愿意来发布几个技能帮我填充内容？
3. 对这个方向有什么看法？

### 正文

**阿里国际站卖家，你们的 AI 焦虑是不是这些？**

——"我知道要用 AI，但不知道用什么提示词"  
——"网上找到的 Prompt 用起来效果一般，针对性不强"  
——"我们公司有个运营用 AI 很厉害，但他离职了，知识就没了"

这是我在和几十个跨境卖家聊的时候，最常听到的三句话。

**问题的本质是：AI 技能是"个人经验"，没有标准化沉淀和共享机制。**

---

**我们的解法：SkillDepot 技能广场**

网址：https://skills.rehomi.com

一句话描述：**把 AI Agent 技能打包成标准格式，发布到市场，任何人都可以下载使用。**

具体怎么运作：

1. **技能包格式**  
   一个 ZIP 文件 + `SKILL.md` 元数据文件。SKILL.md 里面写清楚技能的名字、版本、描述、标签，以及详细的使用说明。

2. **场景分类**  
   目前重点支持的场景标签：
   - 阿里巴巴国际站（AIDC）：Listing 优化、RFQ 自动回复、关键词分析、生意助手集成
   - 亚马逊：广告投放、选品分析、Review 管理、A+ 内容
   - Shopify / 独立站：产品页 SEO、邮件营销自动化、客服机器人
   - WordPress 电商：WooCommerce 自动化
   - 通用跨境：社媒营销（小红书/TikTok/Facebook）、供应链管理、合规风控

3. **质量筛选机制**  
   每个技能有点赞数、下载数、评分。排行榜采用对数衰减防刷分，避免大量低质量技能刷榜，让真正有用的技能浮出水面。

---

**技术细节（掘金读者可能感兴趣）**

- 前端：Next.js 14 App Router + TypeScript + Tailwind CSS
- 后端：NestJS + TypeORM + PostgreSQL
- 存储：阿里云 OSS（技能 ZIP 包）
- 部署：Docker Compose 三容器（Web + API + PostgreSQL）
- 认证：JWT + 微信登录 + 邮箱验证码
- SEO：Server-side sitemap、JSON-LD 结构化数据、双语（中/英）路由

代码开源：https://github.com/jack12w/agent-skill-platform

---
